Go-Maze

A thrilling 3D maze game built with SDL2 and raycasting technology

Clone and Play Go-Maze

Key Features

3D Maze Generation

Go-Maze generates intricate 3D mazes using optimized algorithms, providing a challenging and immersive experience.

Real-time Raycasting

Experience realistic lighting and textures through real-time raycasting techniques, enhancing the gameplay environment.

Interactive Controls

Navigate the maze using intuitive keyboard and mouse controls for a smooth and engaging player experience.

About Go-Maze

This project allowed me to dive into game development using SDL2, Raycasting and C programming.

Development of Go-Maze is part of my Portfolio Project for Holberton School, showcasing my ability to develop interactive and immersive applications.
